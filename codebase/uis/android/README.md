# Android UI

This folder is reserved for the native Android application.

Intended scope:

- primary member-facing mobile experience
- CoverageMatch workflows
- Chapter Community
- ratings and reputation flows
- Prep Lab learning flows
- profile and chapter context screens

The Android Studio project should be created first, then added into this directory.
