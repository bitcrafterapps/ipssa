# UIs

This folder contains all client-facing user interfaces for the IPSSA platform.

Path:

- `codebase/uis/`

Current client surfaces:

- `frontend/`
  Marketing website, BD/lead capture, and limited member portal.

- `ios/`
  Native iOS application.

- `android/`
  Native Android application.

The product is native-mobile-first, so `ios/` and `android/` are the primary operating surfaces. The web frontend remains intentionally narrower in scope.
